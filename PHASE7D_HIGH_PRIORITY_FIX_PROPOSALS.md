## TK-2026-0023（宅建業法・クーリング・オフ）

### 現在の内容

```
question: 自ら売主制限であるクーリング・オフ制度に関する次の記述のうち、正しいものはどれか。

options:
  [0] 「買主が、宅地建物取引業者の『事務所』において買受けの申込みをし、その場で契約を
       締結した場合であっても、契約締結後8日以内であればクーリング・オフによる契約解除ができる。」
  [1] 「買主が、自ら指定した『自宅』において買受けの申込みをし、契約を締結した場合、
       クーリング・オフによる契約解除をすることはできない。」
  [2] 「クーリング・オフによる契約の解除は、買主が宅地建物取引業者に対して口頭で通知
       することによってその効力を生じる。」
  [3] 「クーリング・オフにより契約が解除された場合、売主である宅地建物取引業者は、受領した
       手付金等の全額を速やかに返還しなければならないが、違約金を請求することはできる。」

correct: 1

explanation（現在）:
  「買主が『自ら指定した自宅または勤務先』で申込みや契約を行った場合は、落ち着いて判断
  できる環境とみなされるため、クーリング・オフの適用対象外（解除できない）となります。
  なお、解除の通知は「書面または電磁的記録（※法改正等の動向に注意）」等の確実な方法で
  行う必要があります（口頭は不可）。」
```

### 注意理由

- 正解（correct=1）と各選択肢の正誤判断は正しい
- 問題・選択肢の構造に問題はない
- **問題箇所は解説のみ**：「書面または電磁的記録（※法改正等の動向に注意）」という表現
  - 宅建試験では「クーリング・オフの解除通知は書面（紙）で行う」が出題ポイントの核心
  - 電磁的記録への言及は学習者の混乱を招く
  - 「※法改正等の動向に注意」という注記は不確定な情報として残すべきでない

### 修正案

**correct：変更なし（1のまま）**
**question：変更なし**
**options：変更なし**

**explanation 修正案：**

```
【現在】
「買主が『自ら指定した自宅または勤務先』で申込みや契約を行った場合は、落ち着いて判断
できる環境とみなされるため、クーリング・オフの適用対象外（解除できない）となります。
なお、解除の通知は「書面または電磁的記録（※法改正等の動向に注意）」等の確実な方法で
行う必要があります（口頭は不可）。」

【修正案】
「買主が『自ら指定した自宅または勤務先』で申込みや契約を行った場合は、落ち着いて判断

なお、クーリング・オフによる契約解除の通知は、書面によって行う必要があります（口頭に
```

**変更点の説明：**

## TK-2026-0082（税・その他・住宅ローン控除）

### 現在の内容

```
question: 所得税における住宅借入金等特別控除（住宅ローン控除）に関する次の記述のうち、
           誤っているものはどれか。

options:
  [0] 「住宅ローン控除は、所得金額から差し引く『所得控除』ではなく、算出された税額から
       直接差し引く『税額控除』に分類される。」
  [1] 「この控除を受けるためには、取得した住宅の床面積が一定基準以上であり、かつ、その
       住宅の取得等の日から6ヶ月以内に自らの居住の用に供していなければならない。」
  [2] 「住宅ローン控除の適用を受けようとする最初の年（1年目）は、会社員などの給与所得者
       であっても、年末調整ではなく自ら確定申告をしなければならない。」
  [3] 「返済期間や資金使途に関わらず、どのような金融機関からどのような条件で借り入れた
       ローンであっても、すべて一律に本控除の対象となる。」  ← 誤り（正解）

correct: 3

explanation（現在）:
  「住宅ローン控除を受けるには、償還期間（返済期間）が「10年以上」であることや、特定の
  金融機関等からの借入れであることなど、細かな対象要件が定められており、すべてのローン
  が対象になるわけではありません（※要件の数値は変更されやすいため注意）。」
```

### 注意理由

- 正解（correct=3）は正しい（すべてのローンが対象ではない、は正しい判断）
- 各選択肢の正誤判断も正しい
- **問題箇所は解説のみ**：
  1. 「10年以上」が**返済期間（償還期間）の要件**であることは正しいが、**控除期間**（最大13年）と混同しやすい表現
  2. 「（※要件の数値は変更されやすいため注意）」が漠然としており初学者に何が変わったか不明

### 修正案

**correct：変更なし（3のまま）**
**question：変更なし**
**options：変更なし**

**explanation 修正案：**

```
【現在】
「住宅ローン控除を受けるには、償還期間（返済期間）が「10年以上」であることや、特定の
金融機関等からの借入れであることなど、細かな対象要件が定められており、すべてのローン
が対象になるわけではありません（※要件の数値は変更されやすいため注意）。」

【修正案】
「住宅ローン控除を受けるには、①返済期間（償還期間）が10年以上であること、②取得した
日から6ヶ月以内に居住の用に供すること、③特定の金融機関からの借入れであることなど、
複数の要件を満たす必要があります。また、合計所得金額が2,000万円以下であることも条件
の一つです。すべての住宅ローンが自動的に対象になるわけではありません。」
```

**変更点の説明：**
- 「返済期間10年以上」は**返済期間（償還期間）の要件**であることを①として明示
- 「居住6ヶ月以内」を②として追記（選択肢1の内容を解説にも反映）
- 「特定の金融機関」を③として整理
- 「合計所得金額2,000万円以下」を追記（宅建試験頻出）
- 「※要件の数値は変更されやすいため注意」という不確定表現を削除
- 控除率（0.7%）・控除期間（13年）は税率に関する数値のため、宅建試験の出題範囲外として解説に含めない判断

**verified=true 付与可否案：** THINK判断後・修正確認後に付与を検討

## TK-2026-0095（税・その他・住宅金融支援機構）★最重要

### 現在の内容

```
question: 住宅金融支援機構の証券化支援業務（買取型）の対象となる住宅ローン（いわゆる
           フラット35等）の一般的な融資要件に関する次の記述のうち、誤っているものはどれか。

options:
  [0] 「融資の対象となる住宅は、自ら居住するための住宅、または親族が居住するための住宅
       （セカンドハウス等を含む）でなければならない。」              ← ★ここが問題
  [1] 「買い取り対象となる住宅ローンは、原則として全期間固定金利のローンであり、金利変動
       リスクを機構または投資家が負う仕組みになっている。」
  [2] 「融資対象となる住宅には、床面積の最低基準や、一定の耐久性・断熱性などの適合基準
       が設けられている。」
  [3] 「法人が従業員向けの社宅として購入するための資金の借り入れであっても、法人の信用力
       が高ければ本融資の買い取り対象となる。」                       ← 誤り（現在の正解）

correct: 3（現在）

explanation（現在）:
  「機構の証券化支援（買取型）の対象ローンは、個人が『自ら（または親族が）居住する』ため
  の住宅取得資金に限定されています。法人の社宅購入などの資金は対象外となります。」
```

### 注意理由

- **options[0]「セカンドハウス等を含む」が事実誤認の可能性**
  - フラット35（証券化支援・買取型）の融資対象は「申込本人または親族が居住するための住宅」
  - **セカンドハウス（別荘・週末住宅等）は原則として対象外**（主たる住居であることが条件）
  - 「親族が居住するための住宅」は対象だが、「セカンドハウス等」はこれに含まれない
- options[0]が「正しい記述」を装って「誤り」を含んでいる状態
- ただし現在のcorrect=3（options[3]が誤り）は正しい
  - options[3]「法人社宅購入→高い信用力があれば対象」も誤りであることに変わりはない
- **修正の核心**：options[0]から「（セカンドハウス等を含む）」を削除して「正しい記述」に修正する

### 修正案

**correct：変更なし（3のまま）**
**question：変更なし**

**options[0] 修正案：**

```
【現在】
「融資の対象となる住宅は、自ら居住するための住宅、または親族が居住するための住宅
（セカンドハウス等を含む）でなければならない。」

【修正案】
「融資の対象となる住宅は、申込人本人が居住するための住宅、または申込人の親族が居住
するための住宅でなければならない。」
```

**変更点の説明：**
- 「（セカンドハウス等を含む）」を削除
- 「自ら居住」→「申込人本人が居住」に表現を明確化
- 「親族が居住するための住宅」は正しい要件のため維持
- これにより options[0] が「正しい記述」となり、問題の構造が整合する

**options[1][2][3]：変更なし**

**explanation 修正案：**

```
【現在】
「機構の証券化支援（買取型）の対象ローンは、個人が『自ら（または親族が）居住する』ため
の住宅取得資金に限定されています。法人の社宅購入などの資金は対象外となります。」

【修正案】
「機構の証券化支援（買取型）の対象ローンは、申込人本人またはその親族が居住するための
住宅取得資金（個人による取得）に限定されています。別荘・セカンドハウスや法人による社宅
購入資金は対象外となります。」
```

**変更点の説明：**
- 「別荘・セカンドハウス」が対象外であることを明示（問題の意図を解説で補完）
- 「法人の社宅購入」も対象外であることは維持
