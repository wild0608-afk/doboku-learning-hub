# Takken Learning Hub

宅建試験の個人学習用Webアプリ（MVP版）

---

## ブラウザで開く方法

`index.html` をダブルクリックするか、ChromeやSafariにドラッグ＆ドロップするだけで動作します。
サーバー不要・インストール不要です。

スマホで使う場合は、このフォルダをDropboxやiCloud Driveに置き、
スマホのSafari/Chromeからファイルを開いてください。

---

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリの骨格（HTMLのみ） |
| `style.css` | 全デザイン（緑系・スマホ最優先） |
| `script.js` | 画面描画・操作ロジック・localStorage |
| `questions.js` | 問題データ（ここを編集して問題を追加） |
| `README.md` | このファイル |

---

## 問題を追加する方法

`questions.js` を開いて `QUESTIONS` 配列の末尾に追加します。

```js
{
  id: 9,                       // ← 他と重複しない整数
  category: '宅建業法',         // ← 以下4つのうちどれか
  question: '問題文をここに書く',
  options: [
    '選択肢1',
    '選択肢2',
    '選択肢3',
    '選択肢4'
  ],
  correct: 0,                  // ← 正解のインデックス（0〜3）
  explanation: '解説文をここに書く'
},
```

**使えるカテゴリ（全4種）**
- `'権利関係'`
- `'宅建業法'`
- `'法令上の制限'`
- `'税・その他'`

## 機能一覧

| 機能 | 説明 |
|---|---|
| 分野別学習 | カテゴリを選んで全問を順番に解く |
| ランダムテスト | 全分野からシャッフルして最大10問 |
| 間違い復習 | 過去に1回でも間違えた問題を集中練習 |
| 付箋機能 | 問題画面の☆ボタンで問題をブックマーク |
| 正誤判定 | 選択後すぐに正解と解説を表示 |
| 学習記録 | 全体・分野別の回答数・正答率を確認 |
| localStorage保存 | ブラウザを閉じてもデータが残る |
| リセット | 学習記録画面の下部ボタンで初期化 |

## データの保存場所

ブラウザの `localStorage` に保存されます（キー: `takken_hub_v1`）。
ブラウザのキャッシュをクリアするとデータが消えます。定期的にエクスポートしたい場合は
将来的にJSON出力機能を追加することができます。

## Codexにレビューさせる際の観点

以下の点を重点的に確認してもらうと良いです。

**1. 問題の正確性（questions.js）**
- 各選択肢の内容が現行法令と一致しているか
- `correct` インデックスと解説の正答が一致しているか
- 2024年以降の法改正が反映されているか

**2. ロジックの堅牢性（script.js）**
- `startQuiz` でquestions配列が空の場合のハンドリング
- `answerQuestion` の二重送信防止（`selectedAnswer !== null` ガード）
- `localStorage` 読み書きの try/catch

**3. 拡張性（将来への備え）**
- 問題IDの採番ルール（現在は手動整数）
- カテゴリの追加方法（`CATEGORIES` 配列と `icons` オブジェクトの2か所を更新）
- 1問ずつ正誤を記録している現構造で「直近X回の正答率」が取れるか

**4. スマホUX**
- タップ領域が44px以上確保できているか（iOS HIG基準）
- 長い問題文が画面内に収まるか（端末幅375px基準）
- SafariのSafe Area（ノッチ・ホームインジケーター）対応

**5. アクセシビリティ**
- 色だけに頼らない正誤表示になっているか（アイコン併用）
- disabled状態のボタンのコントラスト比
