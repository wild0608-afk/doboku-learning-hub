### 現在の問題データ

```
category: "宅建業法"
topic: "免許の要件"

question: "宅地建物取引業の免許に関する次の記述のうち、宅地建物取引業法の規定によれば、正しいものはどれか。"

options:
  0: "破産手続開始の決定を受けた個人Aは、復権を得れば、5年を経過していなくても、直ちに免許を受けることができる。"

explanation（現在・57文字）:
「破産者は復権を得れば直ちに欠格事由に該当しなくなり、免許を受けることができます。「5年を経過」する必要はありません。」
```

### Geminiレビュー要約

- 解説が57文字と極めて短く、選択肢1・2・3がなぜ誤りなのか一切説明されていない
- 初学者は「破産したら5年待つ（選択肢1）」「役員に制限なし（選択肢2）」「永遠に受けられない（選択肢3）」を誤解したまま学習を終える可能性が高い
- 解説に「復権すれば直ちにOK」「法人破産の役員も一定期間制限あり」を組み込むことを推奨

### THINK判断：✅ 採用候補

- 破産・復権・法人役員への影響を入れる方向は良い
- 解説拡充の方針は採用
- ただし実装前に現行問題文・選択肢・correctと整合させること

### 実装方針案

以下の3点を解説に盛り込む：

1. **正解（選択肢0）の理由**：破産者は復権を得れば直ちに欠格事由が消え免許可能。「5年待つ」必要はない。
2. **選択肢1の誤り理由**：5年経過は不要。復権の効果は即時。（5年待つのは「禁錮以上の刑」などの場合）
3. **選択肢2の誤り理由**：法人が破産した場合、その法人の役員（「役員であった者」）も欠格事由に該当し、5年間は免許を受けられない（宅建業法5条1項）

**実装前の確認事項**：
- 選択肢2の解釈（法人破産の役員は「5年を経過していない者」として欠格事由に該当）が法令上正確か
- Codexまたは法令原文で「宅建業法5条1項」の条文と照合すること

### 実装後の想定解説文（120〜150文字目標）

```

なお、法人が破産した場合、その法人の役員も「役員であった者」として欠格事由に該当し、
```

## 2. TK-2026-0038：採用候補

### 現在の問題データ

```
id: "TK-2026-0038"
category: "権利関係"
topic: "危険負担"

question: "A・B間でA所有の建物の売買契約を締結した後、引渡し前にその建物が近隣の火災（A・B双方に責任がない事由）により全焼した場合に関する次の記述のうち、民法の規定によれば、正しいものはどれか。"

options:
  0: "建物が滅失したため売買契約は自動的に無効となり、Bは代金を支払う必要はないが、Aに対して損害賠償を請求できる。"
  1: "建物が滅失したため、買主Bは売主Aからの代金支払請求に対して、履行を拒絶（支払を拒否）することができる。"
  2: "買主Bは建物を取得できなくなったとしても、引渡し前の滅失であれば、Aからの代金支払請求を拒むことは一切できない。"
  3: "Aの建物引き渡し義務は消滅しないため、Aは同等の建物を新たに建ててBに引き渡さなければならない。"

correct: 1

explanation（現在・102文字）:
「当事者双方に責任がない事由により履行不能（建物の滅失）となった場合、危険負担のルールにより、買主Bは売主Aからの代金支払請求に対して履行を拒絶することができます。また、Bから契約解除することも可能です。」
```

### Geminiレビュー要約

- 解説は102文字で内容は正確だが、「なぜ代金を払わなくていいのか」という直感的な理解が得られにくい
- 2020年民法改正前は「債権者主義（買主が危険を負担 → 代金を払わなければならなかった）」という旧ルールがあり、受験生が旧法知識と混同するリスクがある
- 選択肢0（自動無効→誤り）・選択肢2（拒めない→誤り）・選択肢3（新たに建てる義務→誤り）の誤り理由が説明されていない

### THINK判断：✅ 採用候補

- 「建物がもらえないなら代金支払いを拒める」と説明する方向は良い
- 2020年改正前後の説明は宅建試験対策として有益
- 現行選択肢との整合確認が必要

### 実装方針案

以下の3点を解説に盛り込む：

1. **正解（選択肢1）の理由**：2020年改正後の危険負担ルール = 双方無責の履行不能では買主は代金支払いを拒絶できる（民法536条）
2. **改正の補足**：旧法（2020年以前）では買主が代金を払わなければならなかった（債権者主義）が、現行法では履行拒絶権が認められる
3. **誤り選択肢の補足**：契約は「自動的に無効」にはならない（選択肢0が誤りの理由）

**実装前の確認事項**：
- 現行法（民法536条）の「債務者主義」への変更が正確に反映されているか
- 「Bから契約解除することも可能」という現行解説の一文は正確か（民法542条：催告なし解除は可能）

### 実装後の想定解説文（120〜160文字目標）

```
2020年の民法改正により、当事者双方に責任のない事由で履行不能（建物全焼）となった場合、
買主Bは売主Aからの代金支払請求を拒絶できます（民法536条）。
「建物をもらえないなら代金は払わなくてよい」と覚えてください。
また、BはAに対して契約解除を求めることもできます。
なお、契約は自動的に無効になるわけではなく、損害賠償請求（選択肢0）も双方無責のため認められません。
```

```
  1: "事務所等以外の場所（喫茶店等）で買受けの申込みをし、その後事務所で契約を締結した場合でも、クーリング・オフができる。"
  2: "クーリング・オフによる解除は、書面でその旨を告知された日から8日以内に書面で行わなければならない。"

```

### Geminiレビュー要約と問題の核心

Geminiは以下を指摘した：

> 「告知された日から8日以内」（選択肢2）は、起算方法によっては正しい表現になる可能性がある。宅建業法では「初日算入」（告知日=1日目として数えて8日間）が適用されるため、「告知された日から8日以内」という表現が正確かどうかを再確認する必要がある。

### THINK判断：⛔ 採用不可・Codex照合が必要

**Gemini案の「電磁的記録」表現は採用しない。**
- 宅建業法のクーリング・オフは、告知書面も解除通知も紙の書面が必要
- 「または電磁的記録」という表現は現行法の要件と異なるため不採用

**以下のより深刻な問題が発覚している：**

### TK-2026-0145 の構造的問題点

#### 問題の構造

| 内容 | 現在の設定 |
|---|---|
| 問題形式 | 「誤っているものはどれか」 |
| correct | 2（選択肢2が誤り＝これが正解） |

| 解説の主張 | 「告知を受けた日を1日目として8日目まで（つまり告知翌日から7日間）」が正しい→選択肢2は誤り |

#### 問題の核心

**「告知された日から8日以内」と「告知日を1日目として8日目まで」は同じ意味になる可能性がある。**

- 宅建業法37条の2では「書面を交付された日から起算して8日を経過した日以後においては、申込みの撤回等を行うことができない」と規定
- 「起算して8日を経過した日以後」= 告知日=1日目として8日目（経過前）まではOK = 初日算入
- 「告知された日から8日以内」も、「から」に告知日を含める（初日算入）と解釈すれば **同じ意味**

#### 解説内の矛盾

現在の解説は「告知翌日から7日間」と説明しているが、これは **初日不算入** の解釈であり、法令上の正確な起算点（初日算入）と矛盾している可能性がある。

- 初日算入の場合：告知日=1日目、8日目まで = **8日間（告知日を含む）**
- 「告知翌日から7日間」は **初日不算入** の読み方

#### 懸念されるシナリオ

選択肢2（「告知された日から8日以内」）が **実は正しい表現** であるとすると：
- correct: 2 は誤り（正しいものを誤りとして問題にしている）
- 問題の構造ごと修正が必要（選択肢の差し替えまたは correct の変更）
